# Masterblog with API

A demo of a blog website with frontend and backend.

# Startup

- Run frontend with  `python3 frontend/frontend_app.py`

- Run backend with `python3 backend/backend_app.py`

- Open in browser http://127.0.0.1:5001/
## Backend

Backend provides APIs.
Can be tested with swagger:
http://127.0.0.1:5002/api/docs
